Generate / Refresh Session

Clicking Open Auth Browser launches a headful Chromium window. Log in to X with Google, then click I've Logged In below.

How It Works
  1. Click Open Auth Browser
  2. A Chrome window opens — log in with your Google account
  3. Once you're on X home, click I've Logged In
  4. The session is saved to auth/session.json
  5. Sessions typically last several weeks
⚠️ auth/session.json contains live login tokens and is gitignored. Never commit it.
